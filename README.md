# Getting started

A simple JS library for building ERC-4337 UserOperations.

# Usage

See the `userop` documentation at [docs.stackup.sh](https://docs.stackup.sh/docs/packages/userop/introduction).

# Contributing

## Prerequisites

- Node 16 or later

## Setup

Install dependencies:

```bash
yarn install
```

Run tests:

```bash
yarn test
```
